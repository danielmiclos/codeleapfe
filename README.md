# CodeLeap Frontend Test

This project is the **Frontend Technical Test for CodeLeap**, built with **React + TypeScript + Vite**.

In addition to implementing the layout provided in the Figma design, **interaction transitions were added using [Motion](https://motion.dev/)** to enhance the user experience with smooth and polished animations.

## Tech Stack

- **React** with **TypeScript**
- **Vite** (using the `react-ts` template)
- **Motion** for interaction transitions and animations

## Getting Started

### Prerequisites

Make sure you have **Node.js** (v18 or higher) and **npm** (or **yarn**) installed on your machine.

### Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/codeleapfe.git
cd codeleapfe
```

2. Install the dependencies:

```bash
npm install
```

### Running the Development Server

```bash
npm run dev
```

The app will be available at `http://localhost:5173` by default.

### Building for Production

```bash
npm run build
```

### Previewing the Production Build

```bash
npm run preview
```
